# CONTEXT.md format

```md
# {Context name}

{One or two sentences describing the bounded context.}

## Language

**Address change request**:
A customer's submitted request to replace a company's registered address.
_Avoid_: Ticket, order
```

- Pick one canonical term and list discouraged synonyms under `_Avoid_`.
- Keep each definition to one or two sentences.
- Define what a term is, not its implementation.
- Include only domain-specific terms.
- Use a single root `CONTEXT.md` for SídloFlow.
